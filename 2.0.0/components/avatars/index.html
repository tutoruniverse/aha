    Avatars

    Introduction#

    An avatar is a circular component, which is used to represent people (askers & experts) and objects (system). Avatars can be photos, illustrations or icons.

    Examples#

    Variants#

    Photo avatar#

    Uploaded by users. When an image hasn't been uploaded, the default avatar will be used.

      <div className="u-inlineBlock u-marginRightMedium">
        <Avatar src="https://raw.githubusercontent.com/gotitinc/aha-assets/master/uifaces/m-10.jpg" />

    Sizes#

    The following are the available sizes of the avatars:

    [
      'extraSmall',
      'small',
      'medium',
      'large',
      'extraLarge',
      'extraLargePlus',
      'huge',
    ].map(item => (
      <div key={item} className="u-inlineBlock u-marginRightMedium">
        <Avatar size={item} src="https://raw.githubusercontent.com/gotitinc/aha-assets/master/gotit/avatar/default.svg" />
      </div>
    )
    );

    Avatar#

    import{ Avatar }from '@ahaui/react'Copy import code for the Avatar component
    NameTypeDefaultDescription
    name
    string

    The Avatar visual name, should be provide via an AssetPlugin with prefix "avatar"

    size
    'extraSmall' | 'small' | 'medium' | 'large' | 'extraLarge' | 'extraLargePlus' | 'huge'
    		'medium'

    Avatar size variants

    src
    string

    Providing a src will render an <img> element

    text
    string

    Providing a text will render an <div> element

    alt
    string
    		'Avatar'

    Providing a alt if src exits

    width
    number

    Set the width of the avatar

    height
    number

    Set the height of the avatar

    as
    elementType
    		<div>

    You can use a custom element type for this component.
